Debian Project (English Version)

This is a project done by me for the college subject "Networks and Operative Systems", given by Universitat Oberta de Catalunya (UOC). This project has consisted about using Debian OS to create a software based on some exercises by using tools like Postgres, Bun web, Docker, and configure HTTP and HTTPs proxies.

It is possible to access to the exercise given by clicking on this link (it is in Spanish): Exercise

1. VM creation and Debian OS installation

On this first exercise we installed the requested Debian OS ISO on a VM (Virtual Machine), which was VirtualBox, as I have experience with this tool.

Nmap and curl tools were downloaded after setting the OS into the VM, as well as Docker engine and Git.
